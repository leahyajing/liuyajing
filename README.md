# README

## ACC102 Personal Assignment
### Topic: FMCG Promotion Effectiveness Analysis

## Project Summary
This project investigates whether promotions are effective in the FMCG sector by analyzing product-related data. The work includes data cleaning, exploratory data analysis, visualization, and interpretation of business insights.

## Submission Files
- `ACC102_Assignment.ipynb` – main notebook containing data cleaning, analysis, and visualizations
- `raw_data.csv` – original dataset
- `cleaned_data.csv` – cleaned dataset used for analysis
- `final_report.pdf` – final report
- `README.md` – project overview and instructions

## Data Cleaning
The following cleaning steps were completed:
- handled missing values
- checked duplicates
- corrected data types
- examined outliers
- prepared variables for analysis

All visualizations and findings in the notebook are based on the cleaned dataset.

## Analysis Conducted
The project includes:
- descriptive statistics
- distribution plots
- boxplots for outlier detection
- category and cluster analysis
- interpretation of business implications

## Key Insight
The analysis shows that product characteristics and category structures vary across the dataset, and these differences may affect how products respond to promotional strategies.
